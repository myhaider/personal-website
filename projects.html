Projects

Political Ads in Social Media

This was a self-directed project in which I conducted exploratory data analysis on Snapchat's Political Ads Library. The dataset contains information about thousands of advertisements from political organizations that were shown to users, so I chose to investigate what features of an ad result in the highest number of views. I built a pipeline using scikit-learn and engineered features to run a linear regression model that predicts an ad's view count. Features included the amount spent on an ad, the number days it was shown, and country of origin.
Read the full study here.

Who Wins Basketball Games?

This was a study done in a small team that analyzed historical data from the NCAA Machine Learning Competiton on Kaggle. Our team investigated the "home-court advantage" phenomena, searching for a possible link between a team's performance and location. We conducted hypothesis testing to verify if teams performed significantly better at home, and built a decision tree model to see if location could even be used as a primary factor in predicting match outcomes.
Full study available on Github.
